WebHooks-總NFT持有人名單

WebHooks-總NFT持有人名單

操作步驟

  1. 開啟新視窗，並且直接輸入總NFT持有人名單API之URL，呼叫蹦彩NFT管理中心之後台伺服器中總NFT持有人名單API功能，等待後台伺服器回應後，瀏覽器會立即下載NFT總持有人名單Excel檔案，點擊Excel檔案後，即可成功取得所要獲得之NFT項目總持有人名單。

補充說明

目前蹦彩NFT管理中心後台伺服器之網址URL非最後使用的版本，在之後部屬上線至伺服器上時，會有正式的URL取代之。
